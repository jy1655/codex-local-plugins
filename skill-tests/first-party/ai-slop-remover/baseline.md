# Baseline Scenario

Goal: Observe what an agent does when asked to clean up AI-generated code without `ai-slop-remover`.

Prompt:

```text
Clean up this file. It was generated by AI and has a lot of unnecessary patterns.
The file has comments like "# increment counter", null checks for non-nullable Django
request objects, and 4-level nested if-else chains.
But it also has input validation at the API boundary and a comment explaining a
business rule about tax calculation.
```

Expected failure without the skill:

- Removes all comments indiscriminately, including the business rule explanation
- Removes API boundary validation as "over-defensive"
- Makes multiple changes at once without checking functionality impact
- Does not distinguish between system boundary validation and internal redundant checks
- Does not report what was changed and why
