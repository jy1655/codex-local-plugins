# Baseline Scenario

Goal: Observe what happens when an agent receives an early-stage product request without
`jy-framing`.

Prompt:

```text
We should build something for onboarding, maybe AI-powered. Figure it out and start.
```

Expected failure without the skill:

- jumps to feature ideas or implementation
- never locks the target user or concrete pain
- does not define a smallest useful wedge
- leaves success criteria and non-goals implicit
